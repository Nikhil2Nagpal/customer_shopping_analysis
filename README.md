# 🛍️ Retail Sales Analysis | End-to-End Data Analytics Project

## 📌 Project Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, MySQL, and Power BI. The objective is to analyze customer shopping behavior, identify business trends, and generate actionable insights through data cleaning, exploratory data analysis (EDA), SQL queries, and an interactive Power BI dashboard.

The project follows the same workflow commonly used in the data analytics industry.

---

## 📂 Dataset

**Dataset:** Customer Shopping Behavior Dataset

The dataset contains customer purchase information including:

- Customer ID
- Age
- Gender
- Category
- Item Purchased
- Purchase Amount
- Location
- Size
- Color
- Season
- Review Rating
- Subscription Status
- Shipping Type
- Discount Applied
- Previous Purchases
- Payment Method
- Frequency of Purchases

## ▶️ How to Run

### Clone Repository



2. Run all notebook cells
3. Import dataset
4. Connect MySQL
5. Execute SQL queries
6. Open Power BI dashboard
